Not bundled with the app
------------------------

The following are used at runtime but are NOT distributed inside the app, so their
licenses do not attach to it:

  - Ollama (MIT) - a separate local app you install; summaries are optional.
  - LLM model weights pulled via Ollama - each carries its own license, which is
    your responsibility (e.g. Qwen2.5 is Apache-2.0; Meta Llama has extra terms).
  - Anthropic Claude API - cloud service used only if you choose it, under
    Anthropic's terms with your own key.
  - Loopback (virtual audio) devices - the optional Obsidian-plugin workflow lets
    you select a loopback input device you already have. None is bundled,
    required, or referenced by the app's code.

The authoritative license text for each project is in its linked repository.
